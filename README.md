## Contributing
このプロジェクトをcloneしてbundle installし、「さぁ開発するぞ」と意気込むところ申し訳ありませんが、以下のコマンドを実行しておいてください。
```shell
bundle exec pre-commit install
git config pre-commit.ruby "bundle exec ruby"
```
